### What I learned

The thing that I learned from this project is how to use custom css properties.

```css

	flex: 1;
	--_background-color: var(--bg-color, black);
	--_icon: var(--icon, black);
	background-color: var(--_background-color);
	color: var(--transparent-white);
}
```
